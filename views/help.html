How to use the website:

There are many different functions on this site and we want to make sure that you feel comfortable when navigating through the page. Please feel free
to use our template pub card and try it out so that you can use the official cards more efficiently. This help card has all the necessary information that
you need to fully enjoy every pub the crawl has to offer. In addition, we recommend that you use these special tools to increase the experience:

  • The voting function helps you to see how long the queue is and you can vote yourself too!
  • The Map page has an overview of all the pubs and where to find them!

About Us
